# Unity To Unity

Unity game prototype developed during the 2018 Indie Game Weekend.

## Status

This repository is archived and no longer maintained. It is preserved as early Unity/C# game-development history.

## Tech Stack

- Unity
- C#

## Running The Project

1. Install a Unity version compatible with the project.
2. Open the repository folder from Unity Hub.
3. Let Unity import assets and regenerate local files.
4. Open the main scene and run it from the editor.

## Notes

- The project may require an older Unity version.
- Generated Unity files and local editor settings may differ by environment.

## Archive Reason

The project is preserved as a game-jam/learning archive while newer Unity and software projects remain active elsewhere.
